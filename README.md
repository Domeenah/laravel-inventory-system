# 📦 Laravel Inventory Management System

A simple Inventory Management System developed using the Laravel framework.

## Features

- Create inventory items
- View inventory items
- Update inventory items
- Delete inventory items
- Bootstrap responsive interface
- MySQL database integration

## Technologies

- Laravel
- PHP
- MySQL
- Bootstrap 5
- HTML
- CSS

## Installation

```bash
git clone https://github.com/Domeenah/laravel-inventory-system.git

cd laravel-inventory-system

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
